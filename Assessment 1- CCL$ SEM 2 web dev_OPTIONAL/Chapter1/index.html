HTML & CSS quick reference

tags, properties, and clean examples — everything covered so far, plus a few extras

📄 HTML tags

<h1> – <h6>

heading elements: <h1> highest (most important) to <h6> lowest.

<h1>Main page title</h1>  
<h2>section heading</h2>
<h3>subsection</h3>

<p>

paragraph block – standard text container.

<p>This is a paragraph of text. It wraps naturally and has spacing above/below.</p>

<a>

anchor / hyperlink. href attribute defines destination.

<a href="https://example.com">visit example</a>  
<a href="#top">back to top</a>

<img>

embeds an image (void element). requires src and alt.

<img src="photo.jpg" alt="description of image">

<ul> & <li>

unordered (bulleted) list. each item inside <li>.

<ul>  
  <li>First item</li>  
  <li>Second item</li>
</ul>

<table> <tr> <th> <td>

tabular data: rows, header cells, data cells.

<table>  
  <tr>    
    <th>Name</th>  
    <th>Age</th>
  </tr>
  <tr>
    <td>Alex</td>  
    <td>28</td>
  </tr>
</table>

<div>

generic block-level container — used for layout and grouping.

<div class="wrapper">  
  <p>content inside a block</p>
</div>

<span>

generic inline container, often used for styling a phrase.

<p>this is <span class="highlight">highlighted</span> text.</p>

<form>

section with interactive controls for user input.

<form action="/submit" method="post">  
  <!-- form elements go here -->  
</form>

<input>

versatile input field. type can be text, email, password, etc.

<input type="text" name="username" placeholder="your name">  
<input type="email" name="email">

<button>

clickable button — can be used in forms or with JavaScript.

<button type="submit">send message</button>  
<button onclick="alert('clicked!')">click me</button>

<label>

associates text with an <input> element (improves accessibility).

<label for="email">email address:</label>  
<input type="email" id="email" name="email">

🎨 CSS properties

color

sets the foreground text color.

p {  
  color: #2c3e50;  
}
.highlight {  
  color: darkred;  
}

background-color

fills element’s background with a color.

div.notice {  
  background-color: #f0f8ff;  
}
button {  
  background-color: #1e90ff;  
}

font-size

specifies size of the font (px, rem, %, em…).

h1 {
  font-size: 2.5rem;  
}
small {
  font-size: 0.875em;  
}

font-family

defines typeface, with fallback stack.

body {
  font-family: 'Segoe UI', Roboto, sans-serif;  
}
code {
  font-family: 'Courier New', monospace;  
}

margin

space outside the element (between elements).

.card {
  margin: 20px;  
}
h2 {
  margin-bottom: 0.5rem;  
}

padding

space inside the element, around its content.

.card {
  padding: 1.5rem;  
}
button {
  padding: 0.6rem 1.2rem;  
}

border

adds a line around the element (width, style, color).

pre {
  border: 1px solid #ccc;  
}
input {
  border: 2px dashed #3498db;  
}

display

changes how an element is rendered (block, inline, flex, grid…).

.card-grid {
  display: grid;  
  gap: 20px;       
}
span.inline-block {
  display: inline-block;  
  width: 100px;
}

text-align

aligns inline content horizontally inside a block.

header {
  text-align: center;  
}
footer p {
  text-align: right;  
}

💡 All code examples use <pre><code> for correct formatting. Indentation is preserved, and tags are escaped with &lt; and &gt; to display literally.
